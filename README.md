# 🍫 Айдана Sweets — Инструкция по запуску

## Структура папок

```
aydana-sweets/
├── server.js          ← сервер (бэкенд)
├── package.json       ← список зависимостей
├── public/            ← ВАШ САЙТ (сюда кладёте все файлы сайта)
│   ├── index-2-2.html
│   ├── style-2.css
│   ├── app.js
│   └── (все картинки .png .webp)
└── admin/             ← АДМИНКА
    ├── api.js         ← уже есть
    ├── login.html     ← уже есть
    ├── dashboard.html ← скопируйте ваш файл
    ├── products.html  ← скопируйте ваш файл
    ├── orders.html    ← уже есть (исправлен)
    └── admin.css      ← скопируйте ваш файл
```

---

## 🚀 Шаг 1 — Установить Node.js

Скачайте с https://nodejs.org (кнопка **LTS**) и установите.

---

## 🚀 Шаг 2 — Подготовить файлы

1. Распакуйте архив `aydana-sweets-server.zip`
2. Скопируйте файлы вашего **сайта** в папку `public/`:
   - `index-2-2.html`
   - `style-2.css`  
   - `app.js` (исправленный — без `s` на строке 23)
   - Все картинки (`.png`, `.webp`)
3. Скопируйте в папку `admin/`:
   - `dashboard.html`
   - `products.html`
   - `admin.css`

---

## 🚀 Шаг 3 — Проверить локально

Откройте терминал (командную строку) в папке проекта:

```bash
npm install
npm start
```

Откройте в браузере: http://localhost:3000  
Админка: http://localhost:3000/admin/login.html

## 🚀 Шаг 4 — Выложить на Railway (бесплатно)

### 4.1 Зарегистрироваться
1. Зайдите на https://railway.app
2. Нажмите **"Login"** → **"Login with GitHub"**
3. Если нет аккаунта GitHub — создайте на https://github.com (бесплатно)

### 4.2 Загрузить проект
1. На Railway нажмите **"New Project"**
2. Выберите **"Deploy from GitHub repo"**
3. Нажмите **"Configure GitHub App"** → разрешите доступ
4. Создайте репозиторий на GitHub и загрузите туда все файлы
5. Выберите этот репозиторий в Railway

### 4.3 Настроить переменные (необязательно, но рекомендуется)
В Railway → ваш проект → вкладка **"Variables"** добавьте:
```
ADMIN_USER = admin
ADMIN_PASS = ваш_надёжный_пароль
JWT_SECRET = любая_длинная_случайная_строка
```

### 4.4 Получить адрес
Railway автоматически запустит сервер и выдаст адрес типа:
`https://aydana-sweets-production.up.railway.app`

Откройте этот адрес — ваш сайт работает в интернете! 🎉

---

## ⚠️ Важно знать

- Данные (заказы, добавленные товары) **хранятся в памяти**. При перезапуске сервера они сбрасываются.
- Если нужно постоянное хранение — напишите, добавим базу данных (SQLite или PostgreSQL).
- Картинки товаров должны лежать в папке `public/` рядом с сайтом.
